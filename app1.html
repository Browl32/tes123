Manual Launch Instructions

If automatic launch doesn't work, you can use the intent URI below with a custom launcher app:

intent://#Intent;action=android.intent.action.MAIN;package=com.Shimofumi_ya.mumasekai;component=com.Shimofumi_ya.mumasekai/com.unity3d.player.UnityPlayerActivity;end

Copy the intent URI above and paste it into an app like "Activity Launcher" or "Shortcut Maker".

See instructions below

Step-by-Step Guide:

  1. Install an app like "Activity Launcher" from Play Store
  2. Open the app and select "Create Shortcut"
  3. Choose "Advanced" or "Intent URI" option
  4. Paste the intent URI in the provided field
  5. Save the shortcut and run it from your home screen
